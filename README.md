# Chocolate-Sales-Analysis-Tableau-Dashboard
A multi-page Tableau dashboard analyzing chocolate sales performance, product profitability, and customer insights across multiple countries and store types.

📌 Project Overview
This project explores a global chocolate sales dataset to uncover trends in revenue, product performance, and customer demographics. The report is structured across 3 dashboard pages:

Sales Overview — Revenue by country, category, and store type
Product Performance — Top products, brand analysis, and profitability by category
Customer Insights — Demographics, segmentation, and revenue by gender
